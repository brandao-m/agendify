# Agendify

Agendify é uma plataforma SaaS para agendamento de serviços.

O sistema permite que profissionais autônomos — como dentistas, barbeiros, esteticistas e outros prestadores de serviço — organizem sua agenda e permitam que clientes agendem horários online.

## Funcionalidades (MVP)

- Cadastro de profissional
- Cadastro de serviços
- Configuração de disponibilidade semanal
- Sistema de agendamento (em desenvolvimento)

## Stack utilizada

Backend:
- Python
- FastAPI
- SQLModel
- PostgreSQL

Segurança:
- Hash de senha com bcrypt

## Objetivo do projeto

Este projeto está sendo desenvolvido como um SaaS real de agendamento, com foco em arquitetura escalável e organização de agenda para profissionais autônomos.

A proposta é evoluir o sistema para permitir que diferentes profissionais utilizem a plataforma para gerenciar seus atendimentos.
